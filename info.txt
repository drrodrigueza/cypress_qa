Cypress

1.- npm init

2.- npm install cypress --save-dev --> Instalación

3.- node_modules/.bin/cypress open

4.- npx cypress open -- npm rum testing

5.- detener consola --> ctrl + c

6.- npm install -D cypress-plugin-tab - para instalar el plugin tab

7.- npm install -D cypress-xpath - Para instalar el plugin de xpath

8.- node_modules/.bin/cypress run - para correr todas las pruebas
8.1- npx cypress run - para correr todas las pruebas
8.2- npx cypress run --headed - para correr todas las pruebas desde el navegador
8.3- npx cypress run --spec "cypress(ruta de ka prueba)" para correr una prueba en específico

9.- npm install --save-dev cypress-file-upload  - Para carga de archivos

10. - npm run cypress:execution - para correr todas las pruebas
11. - node ./cucumber-html-report.js - para generar el reporte

10.- npm install @badeball/cypress-cucumber-preprocessor - instalar cucmber
